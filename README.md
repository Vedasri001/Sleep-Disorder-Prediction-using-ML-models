# Sleep Disorder Prediction Using Machine Learning

## Project Overview
This project predicts sleep disorders using machine learning techniques on a synthetic healthcare dataset. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Performance Evaluation and Comparison

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models
- Random Forest Classifier
- Support Vector Machine (SVM)
- Extra Trees Classifier

## Results
The Random Forest model achieved the highest accuracy of **96.5%** for sleep disorder classification.
